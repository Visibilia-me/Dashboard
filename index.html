Backup dati

I dati sono salvati nel localStorage del browser. Esegui un backup regolare su Google Drive per non perdere nulla. Con Brave configurato per pulire alla chiusura, il backup è obbligatorio.

Istruzioni Google Drive
  1. Clicca Esporta backup JSON — si scarica un file
  2. Apri Google Drive sul tuo browser
  3. Carica il file in una cartella Visibilia/Backup
  4. Ripeti ogni settimana (domenica sera)
💡 Consiglio: imposta un todo ricorrente settimanale "Esegui backup Visibilia" nella sezione Todo.
Brave — escludi dalla pulizia
  1. Apri Brave → Impostazioni
  2. Vai su Privacy e sicurezza
  3. Clicca Cancella dati alla chiusura
  4. Cerca l'opzione Eccezioni siti o Siti che possono sempre...
  5. Aggiungi il tuo dominio Netlify della dashboard
✓ Così localStorage non verrà mai cancellato per la tua dashboard.
